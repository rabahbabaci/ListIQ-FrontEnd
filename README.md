# ListIQ Frontend

Helps users find the best resale platform (Depop, Poshmark, eBay) for their items. Upload a photo, get platform recommendations with pricing, sell probability, and honest verdicts.

**Stack:** React 18, TypeScript, Vite, Tailwind CSS, shadcn/ui

---

## Contributing

### First time setup

```bash
git clone https://github.com/rabahbabaci/listiq-frontend.git
cd listiq-frontend
npm install
npm run dev
```

Open <http://localhost:8080> in your browser. You should see the landing page.

### Pick up an issue

1. Go to the [Issues tab](https://github.com/rabahbabaci/listiq-frontend/issues) on GitHub
2. Browse open issues — they're labeled `code` or `no-code`
3. Comment **"I'll take this"** on the one you want

Every issue includes:

- A description of what to build
- Acceptance criteria (checklist)
- Component props and data shapes
- File paths
- A branch name to use

### Work on it

```bash
git checkout main
git pull
git checkout -b feature/issue-name
```

Use the branch name from the issue (e.g. `feature/worth-it-verdict`).

Do your work — use whatever tools you prefer (ChatGPT, Copilot, Claude, VS Code, etc). Run `npm run dev` to see your changes live in the browser. Follow the file paths and acceptance criteria in the issue.

### Submit your work

```bash
git add .
git commit -m "short description of what you did"
git push origin feature/issue-name
```

Then go to GitHub and open a Pull Request:

- Add a short description of what you built
- Reference the issue (e.g. "Closes #4")
- Rabah will review and merge

### Three rules

1. **Never push to main directly** — everything goes through a PR
2. **One issue per branch** — keep your work focused
3. **Ask in the group chat if you're stuck** — don't spin your wheels alone
